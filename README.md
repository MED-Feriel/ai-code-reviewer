# ⚡ AI Code Reviewer

Plateforme web qui analyse du code automatiquement avec Ollama (LLM local).

## Langages supportés
Python · JavaScript · Dockerfile · CI/CD YAML · Bash · Java · Go

## Lancer en local
```bash
docker compose up --build
```

Accès : http://localhost:3000

## CI/CD
GitHub Actions → Docker Hub automatiquement à chaque push sur main.
